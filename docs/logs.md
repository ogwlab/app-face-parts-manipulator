## Version 7.0.2 (2025-08-04) - 顔形状制御改善 + メントン固定機能

### 🎯 主な変更内容

#### 1. 顔形状制御の改善
- **パラメータ名変更**: `roundness` → `faceShape` に変更し、より直感的な命名に
- **曲率ベースの解剖学的点検出**: ハードコードされた顎角検出を動的な曲率計算に変更
- **実際の顔アスペクト比使用**: 固定値(0.9:1.1)から実際の顔の縦横比を使用するように改善
- **V軸投影アプローチ**: 四角い顔形状のためのより自然な変形アルゴリズム実装

#### 2. メントン（顎先）固定機能の追加
- **新パラメータ**: `fixMenton?: boolean` を ContourParams に追加
- **重み付け反転**: メントン固定時は遠い点ほど大きく変形するように制御
- **段階的減衰**: メントン周辺の点に対して2段階の減衰効果を適用
- **UI連動**: メントン固定時は chinHeight パラメータを自動的に無効化

#### 3. 技術的改善
- **CodeRabbit対応**: すべてのコードレビュー指摘事項を修正
  - 演算子優先順位の明確化（括弧追加）
  - 未使用変数の削除
  - 設定マイグレーション条件の簡潔化
- **LocalStorage設定マイグレーション**: v1.0.0 → v1.1.0 への自動移行
- **TypeScript型安全性**: disabled プロパティの追加など

#### 4. アルゴリズム詳細
- **曲率計算式**: 3点を使用した離散曲率 `crossProduct / (len1 * len2)`
- **解剖学的点検出**: 
  - メントン: 中央付近で最もY座標が大きい点
  - ゴニオン: 曲率の局所最大値を左右で検出
- **ガウシアン重み付け**: `exp(-(distance²) / (2σ²))` による滑らかな遷移
- **アーク長保持**: ローカルエッジ対応の保持による自然な変形

- `src/features/image-warping/forwardMapping/backwardRenderer.ts`

- 全ピクセルに対して三角形包含判定

- **目的**: 品質と処理速度のバランス
- **方式**: エッジ部分のみバックワードマッピング、内部はフォワードマッピング
- **パフォーマンス**: バックワードの1/5〜1/10の処理時間

**実装ファイル**:

#### 4. ファイル名自動生成とダウンロード処理
